# Results

This folder contains all figures generated by the MATLAB implementation of the Gamma-based contact model for functionally graded materials (FGMs).

The results are organized according to the different analyses performed throughout the project.

---

# Figure Groups

## fig_03 – Pressure Distribution

### fig_03_graded_gamma_pressure_distribution
Pressure distribution predicted by the Gamma model for the reference graded material.

Purpose:
- Visualize the contact pressure profile.
- Compare with the classical Hertzian shape.
- Show the influence of elastic grading on stress redistribution.

# Reference Configuration

Unless otherwise specified, results are generated using the reference parameters reported in:

Jitcharoen et al.

Reference point:
- k = 0.497
- E0 = 85.325 GPa mm^-k
- P = 2000 N

---

# Main Physical Findings

The simulations consistently show that:

- Elastic grading increases contact radius.
- Elastic grading reduces maximum contact pressure.
- Pressure redistribution becomes stronger as k increases.
- Lower E0 values promote larger contact areas and lower peak pressures.
- All investigated configurations remain within the analytical validity domain.
- Pressure mitigation can exceed 50% compared with the homogeneous Hertz solution.
