# Documentation

This folder contains documentation related to the project development.

## Reference Paper

The analytical framework implemented in this repository is based on:

Jitcharoen, J., Padture, N. P., Giannakopoulos, A. E., & Suresh, S. (1998).

*Hertzian-Crack Suppression in Ceramics with Elastic-Modulus-Graded Surfaces.*

Journal of the American Ceramic Society, 81(9), 2301–2308.

The original article is not distributed within this repository. Please refer to the publisher's website or your institution's library for access.

## Purpose

The present project reproduces and extends the analytical formulation presented in the reference paper through a MATLAB implementation and a series of parametric investigations on functionally graded materials under Hertzian contact conditions.
